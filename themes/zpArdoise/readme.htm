zpArdoise 1.4.13 – a ZenPhoto/ZenPage theme

Please report bugs on the forum and I will try to fix as soon as possible.

zpArdoise is a theme for ZenPhoto/ZenPage, inherited from fisrt release of zpGalleriffic theme.
Feel free to download and use it, and thanks in advance for your feedback!

Description

Important: To use the release 1.4.13 of the theme, you should have ZenPhoto 1.4.13.x.
If you use an older release of ZenPhoto, see Archives of zpArdoise.


The theme supports the following ZenPhoto plugins:

  • cacheManager, colorbox, comment_form, contact_form, dynamic-locale, favoritesHandler, federated_logon, flag_thumbnail, GoogleMap, image_album_statistics, print_album_menu, rating, register_user, RSS, slideshow (only colorbox option), user_login-out, zenpage

Additional scripts used:

  • Galleriffic (jQuery plugin for rich experience and fast-performing photo galleries)
  • jQuery History Plugin, Slide and Toggle Fade
  • script to navigate with arrow keys on news and images.

Major improvements of 1.4.13 release

  • Simplifications of the theme (deleting frames and rollover effect on thumbnails, size optimization of thumbnails and images, code clean up,...)
  • Some graphical improvements (read changelog.txt)
  • Improvements to be compliant with ZenPhoto 1.4.13 (read changelog.txt)
  • Fix some issues (read changelog.txt)

Installation

  1. Upload the zip file to your computer
  2. Unzip the downloaded zip file locally, and upload the zpArdoise/ folder to the directory /themes/ of your ZenPhoto site
  3. In ZenPhoto administration, go to the Themes tab and activate the zpArdoise theme
  4. Navigate to Options>Theme to view and configure the available options for zpArdoise
  5. If you use an older version of the theme, purge image cache to regenerate it with new sizes of thumbnails and images

Options

  1. You can customize the header image by placing images 900 px wide (900 px - 120 px for example) in the directory /themes/zpArdoise/images/, and configure your header in Options>Theme
  2. You can use the script Galleriffic (presentation script of images).
  3. You can use the RSS icon provided by copying the file /themes/zpArdoise/images/rss.gif in the directory /zp-core/images/

Troubleshooting

  1. If you use Galleriffic script, you should not have any albums that mixes sub-albums and pictures (in this case, there is no navigation bars, so all sub-albums may be not displayed on album.php)

 

 

English version - Version Française

zpArdoise 1.4.13 – un thème ZenPhoto/ZenPage

Rapportez les bugs sur le forum ZenPhoto.org et j’essaierais de les corriger aussitôt que possible.

zpArdoise est un thème pour ZenPhoto/ZenPage hérité de la version initiale du thème zpGalleriffic.
N'hésitez pas à le télécharger et à l'utiliser, et merci d'avance pour vos avis et commentaires !

Description

Important : Pour utiliser la version 1.4.13 du thème, vous devriez utiliser ZenPhoto 1.4.13.x.
Si vous utilisez une ancienne version de ZenPhoto, consultez les Archives de zpArdoise.


Le thème supporte les plugins ZenPhoto suivants :

  • cacheManager, colorbox, comment_form, contact_form, dynamic-locale, favoritesHandler, federated_logon, flag_thumbnail, GoogleMap, image_album_statistics, print_album_menu, rating, register_user, RSS, slideshow (seulement l'option colorbox), user_login-out, zenpage

Scripts utilisés :

  • Galleriffic (plugin jQuery pour des galeries photos rapides à la présentation soignée)
  • jQuery History Plugin, Slide and Toggle Fade
  • script permettant la navigation avec les flèches du clavier sur les actualités et les images.

Améliorations majeures de la version 1.4.13

  • Simplification du thème (suppression des cadres et des effets sur les vignettes, optimisation des tailles des vignettes et des images, nettoyage du code,...)
  • Quelques améliorations graphiques (voir changelog.txt)
  • Améliorations pour être compatible avec les fonctionnalités Zenphoto 1.4.13 (voir changelog.txt)
  • Corrections de quelques bugs (voir changelog.txt)

Installation

  1. Téléchargez le fichier zip sur votre ordinateur
  2. Décompressez le fichier zip localement, et télécharger le dossier zpArdoise/ dans votre répertoire /themes/ de votre site ZenPhoto
  3. Dans l'administration ZenPhoto, allez sur l'onglet Thèmes et activer le thème zpArdoise
  4. Allez sur Options>Thème afin d'afficher et configurer les options disponibles pour zpArdoise
  5. Si vous utilisez une ancienne version du thème, vider le cache d'images afin de le régénérer avec les nouvelles tailles de vignettes et d'images

Options

  1. Vous pouvez personnaliser l'image d’entête en déposant des images d’une largeur de 900 px (900 px - 120 px par exemple) dans le répertoire /themes/zpArdoise/images/, et en configurant le header dans Options>Thème
  2. Vous pouvez utiliser le script Galleriffic (script de présentation des images)
  3. Vous pouvez utiliser l’icône RSS fournie dans le thème en copiant le fichier /themes/zpArdoise/images/rss.png dans le répertoire /zp-core/images/

Problème connu

  1. Si vous utilisez le script Galleriffic, vous devriez éviter d'avoir des albums qui mélangent des sous-albums et des photos (dans ce cas, il n'y a pas de barres de navigations et tous les sous-albums peuvent ne pas être affichés sur album.php)
